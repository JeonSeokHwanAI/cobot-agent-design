# cobot_automation_hub — 멀티 에이전트 설계 발표

> Cobot 사업 PM 업무 자동화 시스템의 Agent · Skill · Orchestrator 아키텍처 소개

## 파일 구성

```
index.html     ← 인터랙티브 슬라이드 (메인)
vercel.json    ← Vercel 배포 설정
```

## 사용 방법

- `←` `→` 키보드 또는 하단 버튼으로 슬라이드 이동
- 우측 점(•) 클릭으로 슬라이드 바로 이동
