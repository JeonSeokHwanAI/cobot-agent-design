Agent vs Skill: 핵심 차이

AGENT

사고하는 존재 (Thinker)

판단상황에 따라 다르게 결정
자율성스스로 계획·실행·평가
반복결과 불충분 시 재시도
협력메인 세션 통해 연계
예시researcher, reviewer, community
VS

SKILL

전문 도구 (Tool)

실행고정된 로직으로 처리
단순입력→처리→출력 (1회성)
경량필요 시에만 로드
전문화특정 도메인에 최적화
예시parse-kakao, search-paper

Skill = Agent의 손  |  Agent = 언제 어떻게 할지 판단하는 존재  |  판단이 필요한가? → Agent / 아니면 → Skill
