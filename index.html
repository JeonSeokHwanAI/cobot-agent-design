Agent vs Skill: 핵심 차이

AGENT

사고하는 존재 (Thinker)

판단
상황에 따라 다르게 결정하고 계획을 수립합니다
자율성
스스로 계획 → 실행 → 평가 루프를 반복합니다
재시도
결과가 불충분하면 스스로 재시도합니다
협력
메인 세션을 통해 다른 Agent와 연계합니다
예시
researcher · reviewer · community
VS

SKILL

전문 도구 (Tool)

실행
고정된 로직으로 정해진 처리만 수행합니다
단순
입력 → 처리 → 출력, 1회성 실행입니다
경량
필요할 때만 컨텍스트에 로드됩니다
전문화
특정 도메인에 최적화된 전문 기능입니다
예시
parse-kakao · search-paper · convert-hwp

Skill = Agent의 손  |  Agent = 언제 어떻게 할지 판단하는 존재  |  판단이 필요한가? → Agent / 아니면 → Skill
